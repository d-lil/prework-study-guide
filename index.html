HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains information about the webpage - metadata
  • The body element represents the visible content shown to the user
  • Title element defines title and is crucial for Search Engine Optimization SEO
  • Style defines CSS styles
  • Base defines base URL
  • Link connects external resource to HTML document
  • Meta defines metadata such as character set, description, keywords, author, and viewport
  • Script embeds client-side scripts in the HTML document
  • Elements listed in other elements are called child elements
  • Body element is the parent element
  • Div creates a division on page
  • Span creates inline division
  • Ul - unordered list, ol - ordered list, li - list
  • Img element contains info about images displayed on webpage
  • Src attribute defines location of image
  • The 'a' element stands for anchor and creates links to the same webpage or other webpages
  • The href attribute points to the URL for the link

CSS

  • Internal style sheet contains CSS rulees for webpage in the head section of HTML file. Rules only apply to that page
  • You can configure CSS classes and IDs to style multiple elements in page code
  • A single change to the CSS rule will apply to all tagged elements
  • External style sheet is most common way to style a webpage
  • Preferred because it allows developers to keep all CSS rules into a separate file
  • When using an external style sheet, link the file using link element placed in HTML's head
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A period preceding card designates a class sector

Git

  • git status: checks what branch we are currently on
  • git bheckout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
